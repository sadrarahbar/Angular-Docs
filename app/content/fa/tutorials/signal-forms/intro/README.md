# یادگیری Angular Signal Forms

این tutorial تعاملی به شما یاد می‌دهد چطور با استفاده از Signal Forms API در Angular، فرم‌های reactive بسازید.

## چطور از این tutorial استفاده کنیم

این tutorial فرض می‌کند مفاهیم اصلی Angular را می‌دانید و با signalها آشنایی پایه‌ای دارید. اگر تازه با Angular آشنا شده‌اید، [راهنمای essentials](/essentials) ما را بخوانید. اگر تازه با signalها آشنا شده‌اید، ابتدا [tutorial مربوط به signalها](/tutorials/signals) را کامل کنید.

هر step یک مفهوم در Signal Forms را نشان می‌دهد. شما یک فرم login کامل را از صفر می‌سازید و مبانی را مرحله‌به‌مرحله یاد می‌گیرید.

**مسیر یادگیری شما:**

1. راه‌اندازی form model با TypeScript و signalها
2. اتصال فرم به template
3. اضافه کردن ruleهای validation
4. نمایش errorهای validation به کاربران
5. مدیریت form submission
6. بررسی topicهای پیشرفته و قدم‌های بعدی

اگر جایی گیر کردید، بالای صفحه روی "Reveal answer" کلیک کنید.

خیلی خب، بیایید [شروع کنیم](/tutorials/signal-forms/1-set-up-form-model)!
