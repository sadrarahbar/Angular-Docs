# یادگیری Angular signals

این tutorial تعاملی، مبانی Angular signals و نحوه استفاده از آن‌ها برای ساخت برنامه‌های reactive را به شما آموزش می‌دهد.

## چطور از این tutorial استفاده کنیم

این tutorial فرض می‌کند که مفاهیم اصلی Angular را می‌دانید. اگر تازه با Angular آشنا شده‌اید، [راهنمای essentials](/essentials) ما را بخوانید.

هر step یک مفهوم در Angular signals را نشان می‌دهد. می‌توانید یکی از آن‌ها یا همه آن‌ها را انجام دهید.

اگر جایی گیر کردید، بالای صفحه روی "Reveal answer" کلیک کنید.
