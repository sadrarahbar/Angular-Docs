# به tutorial Angular خوش آمدید

این tutorial تعاملی، building blockهای پایه را به شما یاد می‌دهد تا ساخت برنامه‌های عالی با Angular را شروع کنید.

## چطور از این tutorial استفاده کنیم

برای فهمیدن Angular، لازم است آشنایی پایه‌ای با HTML، CSS و JavaScript داشته باشید.

هر step یک مفهوم در Angular را نشان می‌دهد. می‌توانید یکی از آن‌ها یا همه آن‌ها را انجام دهید.

اگر جایی گیر کردید، بالای صفحه روی "Reveal answer" کلیک کنید.

خیلی خب، بیایید [شروع کنیم](/tutorials/learn-angular/1-components-in-angular).
