## پشتیبانی مرورگر

انگولار برای تعریف پشتیبانی مرورگر از Baseline با وضعیت [«به‌طور گسترده در دسترس»](https://web.dev/baseline) استفاده می‌کند.
برای هر نسخه major، انگولار از مرورگرهای موجود در Baseline یک تاریخ انتخاب‌شده نزدیک به تاریخ انتشار همان نسخه major پشتیبانی می‌کند.

Baseline با وضعیت «به‌طور گسترده در دسترس» شامل مرورگرهایی است که کمتر از ۳۰ ماه \(۲.۵ سال\) پیش از تاریخ انتخاب‌شده در مجموعه مرورگرهای اصلی Baseline \(Chrome، ‏Edge، ‏Firefox و Safari\) منتشر شده‌اند و تقریباً ۹۵ درصد کاربران وب را پوشش می‌دهد.

## Polyfillها

انگولار بر پایه جدیدترین استانداردهای پلتفرم وب ساخته شده است.
پشتیبانی از دامنه گسترده مرورگرها دشوار است، زیرا همه آن‌ها از تمام قابلیت‌های مرورگرهای مدرن پشتیبانی نمی‌کنند.
برای جبران این موضوع، scriptهای polyfill را برای مرورگرهایی که باید پشتیبانی شوند بارگذاری می‌کنید.
دستورالعمل افزودن polyfillها به پروژه در ادامه آمده است.

IMPORTANT: ‏polyfillهای پیشنهادشده مواردی هستند که application کامل انگولار را اجرا می‌کنند.
ممکن است برای قابلیت‌هایی که در این فهرست نیستند به polyfillهای بیشتری نیاز داشته باشید.

HELPFUL: ‏Polyfillها نمی‌توانند به شکلی جادویی یک مرورگر قدیمی و کند را به مرورگری مدرن و سریع تبدیل کنند.

## فعال کردن polyfillها در پروژه‌های CLI

[Angular CLI](tools/cli) از polyfillها پشتیبانی می‌کند.
اگر پروژه را با CLI ایجاد نمی‌کنید، [دستورالعمل polyfill برای کاربران غیر CLI](#polyfills-for-non-cli-users) را ببینید.

گزینه‌های `polyfills` در [browser و test builder](tools/cli/cli-builder) می‌توانند مسیر کامل یک فایل \(برای نمونه `src/polyfills.ts`\)، مسیر نسبی نسبت به workspace فعلی یا module specifier \(برای نمونه `zone.js`\) باشند.

اگر فایل TypeScript ایجاد می‌کنید، حتماً آن را در property به نام `files` در فایل `tsconfig` قرار دهید.

```json

{
  "extends": "./tsconfig.json",
  "compilerOptions": {
    ...
  },
  "files": [
    "src/main.ts",
    "src/polyfills.ts"
  ]
  ...
}
```

## Polyfillها برای کاربران غیر CLI

اگر از CLI استفاده نمی‌کنید، scriptهای polyfill را مستقیماً به صفحه میزبان وب \(`index.html`\) اضافه کنید.

<script src="node_modules/zone.js/bundles/zone.umd.js"></script>
<!-- application polyfills -->
