# importهای Standalone استفاده‌نشده

این diagnostic حالت‌هایی را تشخیص می‌دهد که آرایه `imports` یک `@Component` شامل symbolهایی است که داخل template استفاده نشده‌اند.

```typescript
@Component({
  imports: [UsedDirective, UnusedPipe],
})
class AwesomeCheckbox {}
```

## مشکل این چیست؟

importهای استفاده‌نشده noise غیرضروری به کد شما اضافه می‌کنند و می‌توانند زمان compilation را افزایش دهند.

## به جای آن چه کار کنم؟

import استفاده‌نشده را حذف کنید.

```typescript
@Component({
  imports: [UsedDirective],
})
class AwesomeCheckbox {}
```

## اگر نتوانم از این حالت اجتناب کنم چه؟

این diagnostic را می‌توان با ویرایش فایل `tsconfig.json` پروژه غیرفعال کرد:

```json
{
  "angularCompilerOptions": {
    "extendedDiagnostics": {
      "checks": {
        "unusedStandaloneImports": "suppress"
      }
    }
  }
}
```

برای اطلاعات بیشتر، [پیکربندی extended diagnostic](extended-diagnostics#configuration) را ببینید.
