# نبود siblingها در Hydration

این error شبیه [error NG0500](errors/NG0500) است، اما نشان می‌دهد که فرآیند hydration در Angular انتظار داشته در یک نقطه مشخص از ساختار DOM، siblingهای بیشتری وجود داشته باشند. برای اطلاعات بیشتر به صفحه [error NG0500](errors/NG0500) مراجعه کنید.

## debug کردن error

برای اطلاعات بیشتر درباره روش debug کردن این error، [error NG0500](errors/NG0500) را ببینید.
