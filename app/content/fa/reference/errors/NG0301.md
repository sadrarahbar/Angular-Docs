# Export پیدا نشد

<docs-video src="https://www.youtube.com/embed/fUSAg4kp2WQ"/>

Angular نمی‌تواند directiveای با export name مربوط به `{{ PLACEHOLDER }}` پیدا کند. export name در property مربوط به `exportAs` در decorator مربوط به directive مشخص می‌شود. این حالت هنگام استفاده از FormsModule یا moduleهای Material در templateها رایج است، وقتی import کردن moduleهای متناظر را فراموش کرده باشید.

HELPFUL: این معادل runtime برای error رایج compiler یعنی [NG8003: No directive found with export](errors/NG8003) است.

## debug کردن error

از export name استفاده کنید تا templateها یا moduleهایی را که از این export استفاده می‌کنند trace کنید.

مطمئن شوید همه dependencyها به درستی در NgModuleهای شما import و declare شده‌اند. برای مثال، اگر export پیدا نشده `ngForm` است، باید `FormsModule` را import کنیم و آن را در لیست imports در `*.module.ts` declare کنیم تا error حل شود.

```ts

import { FormsModule } from '@angular/forms';

@NgModule({
…
imports: [
FormsModule,
…

```

اگر اخیراً importای اضافه کرده‌اید، ممکن است لازم باشد server خود را restart کنید تا این تغییرات دیده شوند.
