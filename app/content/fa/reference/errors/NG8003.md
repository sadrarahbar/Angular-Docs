# target مربوط به reference پیدا نشد

<docs-video src="https://www.youtube.com/embed/fUSAg4kp2WQ"/>

Angular نمی‌تواند directiveای با export name مربوط به `{{ PLACEHOLDER }}` پیدا کند. این حالت معمولاً به خاطر missing import یا نبودن [`exportAs`](api/core/Directive#exportAs) روی یک directive رخ می‌دهد.

HELPFUL: این معادل compiler برای error رایج runtime یعنی [NG0301: Export Not Found](errors/NG0301) است.

## debug کردن error

از نام string مربوط به export پیدا نشده استفاده کنید تا templateها یا moduleهایی را که از این export استفاده می‌کنند trace کنید.

مطمئن شوید همه dependencyها به درستی import و در Moduleهای ما declare شده‌اند. برای مثال، اگر export پیدا نشده `ngForm` است، باید `FormsModule` را import کنیم و آن را در لیست imports در `*.module.ts` declare کنیم تا error مربوط به missing export حل شود.

```ts

import { FormsModule } from '@angular/forms';

@NgModule({
…
imports: [
FormsModule,
…

```

اگر اخیراً importای اضافه کرده‌اید، باید server خود را restart کنید تا این تغییرات دیده شوند.
