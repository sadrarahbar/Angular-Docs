# element ریشه پیدا نشد

کامپوننت‌های bootstrap شده در property مربوط به `bootstrap` در decorator مربوط به `@NgModule` تعریف می‌شوند، یا برای کامپوننت‌های standalone به عنوان اولین پارامتر `bootstrapApplication` قرار می‌گیرند.

این error زمانی رخ می‌دهد که Angular تلاش می‌کند یکی از این کامپوننت‌ها را bootstrap کند، اما node متناظر آن را در DOM پیدا نمی‌کند.

## debug کردن error

این مشکل زمانی رخ می‌دهد که selector با tag مطابقت نداشته باشد:

```typescript
@Component({
  selector: 'my-app',
  ...
})
export class App {}
```

<!-- prettier-ignore -->
```html
<html>
  <app-root></app-root> <!-- Doesn't match the selector -->
</html>
```

tag را با مورد درست جایگزین کنید:

<!-- prettier-ignore -->
```html
<html>
  <my-app></my-app> <!-- OK -->
</html>
```
