# multi provider نامعتبر

Angular runtime زمانی این error را throw می‌کند که tokenای را inject کند که قرار بوده با `multi: true` استفاده شود، اما به جای آن یک مقدار غیر Array پیدا شود. برای مثال، `ENVIRONMENT_INITIALIZER` باید به این شکل provide شود: `{provide: ENVIRONMENT_INITIALIZER, multi: true, useValue: () => {...}}`.
