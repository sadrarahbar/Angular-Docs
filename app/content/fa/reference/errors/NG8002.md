# Attribute نامعتبر

<docs-video src="https://www.youtube.com/embed/wfLkB3RsSJM"/>

یک attribute یا property در زمان compilation قابل resolve شدن نیست.

این error زمانی ایجاد می‌شود که تلاش کنید به propertyای bind کنید که وجود ندارد. هر property binding باید با یکی از موارد زیر مطابقت داشته باشد:

- یک property native روی element HTML، یا
- یک property از نوع `input()`/`@Input()` مربوط به کامپوننت یا directive اعمال‌شده روی element.

error مربوط به runtime برای این حالت `NG0304: '${tagName}' is not a known element: …'` است.

## debug کردن error

مستندات مربوط به [binding syntax](guide/templates) مشخصی را که استفاده کرده‌اید ببینید. این مشکل معمولاً typo یا import نادرست است. همچنین ممکن است directiveای با property selector به نام 'name' یا input مربوطه وجود نداشته باشد.
