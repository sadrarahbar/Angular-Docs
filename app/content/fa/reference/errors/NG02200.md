# Iterable Differ پیدا نشد

`NgFor` نتوانسته برای مقدار پاس‌داده‌شده یک iterable differ پیدا کند. مطمئن شوید مقدار iterable است، مثل یک `Array`.

## debug کردن error

وقتی در template از ngFor استفاده می‌کنید، باید از نوعی Iterable مثل `Array`، `Set`، `Map` و موارد مشابه استفاده کنید. اگر می‌خواهید روی keyهای یک object تکرار انجام دهید، بهتر است به جای آن [KeyValue pipe](/api/common/KeyValuePipe) را ببینید.
