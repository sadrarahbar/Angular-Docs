# selector نامعتبر برای Shadow DOM

selector کامپوننتی که از `ViewEncapsulation.ShadowDom` استفاده می‌کند، با الزامات نام tag مربوط به custom element مطابقت ندارد.

برای اینکه یک tag name به عنوان نام معتبر custom element در نظر گرفته شود، باید:

- با حروف کوچک باشد.
- شامل hyphen باشد.
- با یک حرف شروع شود \(a-z\).

## debug کردن error

selector کامپوننت خود را تغییر نام دهید تا با این الزامات مطابقت داشته باشد.

**قبل:**

```ts
@Component({
  selector: 'comp',
  encapsulation: ViewEncapsulation.ShadowDom
…
})

```

**بعد:**

```ts
@Component({
  selector: 'app-comp',
  encapsulation: ViewEncapsulation.ShadowDom
…
})
```
