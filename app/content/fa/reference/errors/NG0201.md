# Provider پیدا نشد

<docs-video src="https://www.youtube.com/embed/lAlOryf1-WU"/>

وقتی تلاش می‌کنید یک service را inject کنید اما provider متناظر آن را declare نکرده‌اید، این error را می‌بینید. provider یک mapping است که مقداری را فراهم می‌کند که می‌توانید آن را داخل constructor یک class در برنامه خود inject کنید.

درباره providerها در [راهنمای Dependency Injection](guide/di) بیشتر بخوانید.

## debug کردن error

از objectای که error می‌گوید provider آن missing است به عقب برگردید: `No provider for ${this}!`. این error معمولاً در serviceهایی throw می‌شود که providerهای موجود ندارند.

برای رفع error، مطمئن شوید service شما در لیست providerهای یک `NgModule` ثبت شده یا decorator مربوط به `@Service` را در بالای خود دارد.

رایج‌ترین راه‌حل اضافه کردن provider در `@Service` است:

```ts
@Service()
```
