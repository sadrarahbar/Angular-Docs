# پشتیبانی JSONP در پیکربندی HttpClient

Angular زمانی این error را تولید می‌کند که بدون فراهم کردن پشتیبانی لازم در پیکربندی `HttpClient`، تلاش کنید یک request از نوع `JSONP` انجام دهید. برای فعال کردن پشتیبانی `JSONP`، می‌توانید یکی از کارهای زیر را انجام دهید:

- هنگام استفاده از `bootstrapApplication`، `withJsonpSupport()` را به عنوان argument در فراخوانی تابع `provideHttpClient` اضافه کنید؛ برای مثال `provideHttpClient(withJsonpSupport())`
- هنگام استفاده از bootstrap مبتنی بر NgModule، `HttpClientJsonpModule` را در AppModule ریشه import کنید.

## debug کردن error

مطمئن شوید پشتیبانی JSONP یا با فراخوانی تابع `withJsonpSupport`، وقتی از `provideHttpClient` استفاده می‌شود، یا با import کردن module مربوط به `HttpClientJsonpModule` همان‌طور که بالا توضیح داده شد، به برنامه شما اضافه شده باشد.
