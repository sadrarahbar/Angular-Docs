# dependencyهای `@defer` بارگذاری نشدند

این error زمانی رخ می‌دهد که بارگذاری dependencyهای یک block مربوط به `@defer` شکست بخورد، معمولاً به خاطر شرایط ضعیف شبکه، و هیچ block مربوط به `@error` برای مدیریت حالت شکست پیکربندی نشده باشد. نبودن block مربوط به `@error` در این سناریو ممکن است تجربه کاربری ضعیفی ایجاد کند.

## debug کردن error

بررسی کنید که برای مدیریت حالت‌های شکست، blockهای `@error` را به blockهای `@defer` خود اضافه کرده باشید.
