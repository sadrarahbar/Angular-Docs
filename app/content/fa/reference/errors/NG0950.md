# input ضروری قبل از set شدن مقدار خوانده شده است.

یک input ضروری خوانده شده، اما هیچ مقداری به آن bind نشده است.

این می‌تواند زمانی رخ دهد که یک input ضروری در directive یا کامپوننت شما خیلی زود خوانده شود. این حالت معمولاً وقتی اتفاق می‌افتد که input به عنوان بخشی از ساخت class خوانده شود.

inputها تضمین می‌شوند که از lifecycle hook مربوط به `ngOnInit` به بعد در دسترس باشند.

## رفع error

input ضروری را در contextهای reactive بخوانید؛ برای مثال، در خود template، داخل یک `computed` یا داخل یک effect.

به عنوان راه جایگزین، input را داخل lifecycle hook مربوط به `ngOnInit` یا بعد از آن بخوانید.
