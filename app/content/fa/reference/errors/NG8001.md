# Element نامعتبر

یک یا چند element در زمان compilation قابل resolve شدن نیستند، چون element در HTML spec تعریف نشده یا هیچ کامپوننت یا directiveای با چنین element selectorای وجود ندارد.

HELPFUL: این معادل compiler برای error رایج runtime یعنی `NG0304: '${tagName}' is not a known element: ...` است.

## debug کردن error

از نام element در error استفاده کنید تا فایل‌هایی را که element در آن‌ها استفاده شده پیدا کنید.

بررسی کنید که name و selector درست باشند.

مطمئن شوید کامپوننت به درستی داخل NgModule یا کامپوننت standalone شما import شده است؛ این را با بررسی حضور آن در field مربوط به `imports` چک کنید. اگر کامپوننت در یک NgModule declare شده، یعنی standalone نیست، مطمئن شوید از همان NgModule به درستی export شده است؛ این را با بررسی حضور آن در field مربوط به `exports` چک کنید.

هنگام استفاده از custom elementها یا web componentها، مطمئن شوید [`CUSTOM_ELEMENTS_SCHEMA`](api/core/CUSTOM_ELEMENTS_SCHEMA) را به application module اضافه کرده‌اید.

اگر این کار error را حل نکرد، libraryهای import شده را برای تغییرات اخیر در exportها و propertyهایی که استفاده می‌کنید بررسی کنید و server خود را restart کنید.
