# Token پیدا نشد

در زمان compile، هیچ injection tokenای برای یک پارامتر constructor وجود ندارد. [InjectionTokenها](api/core/InjectionToken) tokenهایی هستند که می‌توانند در یک Dependency Injection Provider استفاده شوند.

## debug کردن error

به پارامتری که error را ایجاد می‌کند و همه استفاده‌های class نگاه کنید. این error معمولاً زمانی throw می‌شود که constructor پارامترهایی با typeهای primitive مثل `string`، `number`، `boolean` و `Object` تعریف کند.

از method مربوط به `@Injectable` یا decorator مربوط به `@Inject` از `@angular/core` استفاده کنید تا مطمئن شوید typeای که inject می‌کنید reified است، یعنی representation در runtime دارد. مطمئن شوید به این decorator یک provider اضافه کرده‌اید تا [NG0201: No Provider Found](errors/NG0201) throw نشود.
