# Pipe پیدا نشد

<docs-video src="https://www.youtube.com/embed/maI2u6Sxk9M"/>

Angular نمی‌تواند pipeای با این نام پیدا کند.

[pipe](guide/templates/pipes) ارجاع‌داده‌شده در template به درستی نام‌گذاری یا declare نشده است.

برای استفاده از pipe:

- مطمئن شوید نام استفاده‌شده در template با نام تعریف‌شده در decorator مربوط به pipe مطابقت دارد.
- pipe را به آرایه `imports` کامپوننت خود اضافه کنید یا اگر pipe مقدار `standalone: false` دارد، `NgModule`ای را که pipe به آن تعلق دارد اضافه کنید.

## debug کردن error

از نام pipe استفاده کنید تا محل declare شدن و استفاده شدن pipe را trace کنید.

برای رفع این error:

- اگر pipe محلیِ `NgModule` است، در decorator مربوط به pipe یک نام یکتا به آن بدهید و آن را در `NgModule` declare کنید.
- اگر pipe standalone است یا در `NgModule` دیگری declare شده، آن را به field مربوط به `imports` در کامپوننت standalone یا `NgModule` فعلی اضافه کنید.

اگر اخیراً import یا declarationای اضافه کرده‌اید، ممکن است لازم باشد server خود را restart کنید تا این تغییرات دیده شوند.
