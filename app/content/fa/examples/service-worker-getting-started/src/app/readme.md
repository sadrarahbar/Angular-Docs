# دستورالعمل‌های دانلود نمونه Angular Universal

این sample code دانلودشده برای [راهنمای Angular Universal (Standalone)](https://angular.dev/guide/ssr) است.

## نصب و اجرا

1. `npm install` برای نصب packageهای `node_module`
2. `npm run dev:ssr` برای راه‌اندازی server و application
3. مرورگر را روی `http://localhost:4200` باز کنید
